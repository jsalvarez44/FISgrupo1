# FISgrupo1
Grupo 1 de Fundamentos de Ingenieria de Software

Los 'files' presentes son ejercicios y teoria de las clases de Fundamentos de Programacion de la Universidad de las Fuerzas Armadas ESPE
Los ejercicios estan en lenguaje C de programacion probados en el compilador de Dev C++
Este proyecto de github tiene como finalidad llevar un registro del codigo fuente de los programas hechos en clase, debere y el proyecto
El proyecto refiere a la creacion de un juego estilo pacman de nuestra autoria (Carlos y mia)
